# Instructions

run ```pip install -r requirements.txt```

Create a ```config.py```

in the ```config.py``` you will have 4 variables:

```USER_ID='your_id'```

```GAME_ID='gameid'```

```USER_NAME='botid'```

```STORE_REPLAY=True```

change these variables for your personal bot

you do not need to make the ```replays.txt``` file
